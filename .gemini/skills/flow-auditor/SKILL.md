# Flow Auditor

This skill provides a structured workflow for recurring website audits.

## Audit Workflow

### 1. Navigation & Flow Check
- Verify all links in Navigation.tsx lead to the correct pages.
- Check App.tsx and HomePage.tsx for logical entry points.

### 2. UI/UX Consistency
- Audit theme.css and components for adherence to design tokens.

### 3. Technical Health
- Run npm run lint and check for type errors in src/.
